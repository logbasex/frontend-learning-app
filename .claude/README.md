# .claude Configuration

This folder contains configuration files for Claude Code (claude.ai/code) to ensure consistent implementation standards across the project.

## Files

### instructions.md
Comprehensive implementation guidelines for Claude Code, including:
- **Language requirements** (English only)
- Code style standards
- Module implementation patterns
- Component examples
- Quality checklist

### preferences.json
Machine-readable preferences for automated tooling:
- Language setting (English)
- Reference files
- Implementation rules

## Purpose

These files ensure that:
1. All code is written in English (mandatory requirement)
2. New modules follow the established pattern
3. Code quality and consistency is maintained
4. Claude Code has clear guidance for implementing new features

## Usage

When Claude Code works on this repository, it will automatically read these files and follow the guidelines specified here.

## Do Not Delete

These configuration files are tracked in git and should be committed. They are part of the project's development standards.
